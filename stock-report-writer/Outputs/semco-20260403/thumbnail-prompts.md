# 삼성전기 썸네일 생성 프롬프트

## 쇼츠 썸네일 (thumbnail-shorts.png)
**사이즈**: 1080x1920 (세로형)

```
Create a YouTube Shorts thumbnail (1080x1920px, vertical).

Content:
- Main text (very large, bold): "삼성전기"
- Sub text (large, red): "목표주가 59만원"
- Highlight (biggest, bright red): "▲ +41.5%"
- Small bottom text: "유진투자증권 리포트 | 2026.04.03"

Design:
- Dark background (deep navy #0D1B2A or black)
- "삼성전기" in white, very large at top
- Target price "590,000원" in bright red, massive size center
- Upside "+41.5%" with up arrow in even brighter red/orange
- Clean, minimal design - numbers must pop on small mobile screen
- Optional: subtle upward arrow graphic in background
- No clutter, maximum impact
```

```
Create a YouTube thumbnail (1280x720px, horizontal).

Content:
- Left side title: "목표주가\n64% 상향"
- Stock badge: "삼성전기 009150"

- Upside badge: "+41.5% ▲"
- Bottom: "유진투자증권 리포트"

Design:
- Split layout: left dark navy, right slightly lighter navy
- Left title text in white, bold, large
- Right side target price in bright red, massive
- Upside badge in red with white text
- Stock name badge with rounded corners
- Professional financial thumbnail style
- High contrast, readable from distance
- No misleading expressions like "무조건 오른다"
```
