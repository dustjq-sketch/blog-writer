# 리포트 기획서 히스토리 (Report Briefs)

각 종목별 분석 기획서(brief.md)의 요약을 누적 관리하는 파일입니다.
내부 링크 연결, 후속 콘텐츠 기획, 중복 내용 방지에 활용됩니다.

---

## 기획서 요약 기록

분석이 완료될 때마다 아래 형식으로 추가됩니다:

```
---
## [날짜] - [종목명] ([티커])
- **slug**: 
- **핵심 앵글**: 
- **메인 키워드**: 
- **핵심 포인트 요약**:
  1. 
  2. 
  3. 
- **연관 종목**: (내부 링크 기회)
---
```

---

## 내부 링크 맵

종목 간 연관 관계를 정리하여 내부 링크 전략에 활용:

| 종목 A | 종목 B | 연관 관계 | 링크 방향 |
|--------|--------|-----------|-----------|
| (분석 후 자동 누적) | | | |

---

*이 파일은 Report Orchestrator와 Content Reviewer가 자동으로 업데이트합니다.*
