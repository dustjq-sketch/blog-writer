# 콘텐츠 검수 보고서 (Content Review)

Content Reviewer Agent가 검수할 때마다 누적으로 기록하는 파일입니다.

---

## 검수 보고서 기록

검수가 완료될 때마다 아래 형식으로 추가됩니다:

```
---
## 검수 보고서 - YYYY-MM-DD

### 분석 종목
- [종목명A] (slug-a)
- [종목명B] (slug-b)

### 팩트 체크 결과
- [수정 사항 있을 경우 기재]
- 이상 없음

### SEO 점검 결과
| 종목 | 제목길이 | 키워드밀도 | H태그 | 이미지alt | 내부링크 | 면책문구 | 글자수 |
|------|---------|-----------|-------|----------|---------|---------|--------|
| [종목A] | ✓ | ✓ | ✓ | ✓ | ✓ | ✓ | ✓ |
| [종목B] | ✓ | ✓ | ✓ | ✓ | ✓ | ✓ | ✓ |

### 유튜브 스크립트 점검
- [종목A]: 이상 없음
- [종목B]: [수정 사항 있을 경우]

### 수정 사항
- [수정 내역]

### 발행 전략
**추천 발행 순서:**
1. [종목명A] - [이유]
2. [종목명B] - [이유]

**발행 권장 시간:**
- 블로그: 평일 오전 7~9시 or 오후 6~8시
- 유튜브: 평일 오전 8~9시 (장 시작 전) or 오후 4~6시 (장 마감 후)

**주의사항:**
- [특이사항 있을 경우]
---
```

---

*이 파일은 Content Reviewer Agent가 자동으로 업데이트합니다.*
