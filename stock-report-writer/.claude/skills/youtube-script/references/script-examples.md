# 유튜브 풀영상 스크립트 예시

내 풀영상 스타일을 재현하기 위한 참조 파일입니다.
실제 내가 올린 영상 스크립트를 여기에 추가하세요.

---

## 후크 패턴 모음 (풀영상용)

```
"[종목명] [증권사] 리포트 나왔습니다. 오늘 핵심만 정리해드릴게요."
"[증권사]가 [종목명] 목표주가를 OOO원으로 올렸어요. 이유가 뭔지 같이 봐봅시다."
"[종목명] 리포트, 5분 안에 다 정리됩니다."
```

---

## 섹션 전환 멘트

- "자, 이유 들어가볼게요."
- "두 번째입니다."
- "좋은 얘기만 하면 섭섭하죠, 리스크도 짚어볼게요."
- "정리할게요."

---

## 클로징 문구 모음

```
"오늘 [종목명] 리포트 정리 어떠셨나요?
쇼츠 버전도 올려뒀으니 짧게 보고 싶은 분들은 참고하세요.
투자 결정은 항상 여러분이 직접 하세요.
구독과 좋아요 부탁드립니다!"
```
