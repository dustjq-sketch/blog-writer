# 유튜브 스크립트 예시

유튜브 스크립트 작성 시 참조할 예시 모음입니다.
실제 내가 사용한 스크립트나 마음에 드는 구성을 여기에 추가하세요.

---

## 효과적인 후크 (Hook) 예시 모음

### 숫자 후크
```
"증권사 OO곳 중 OO곳이 이 종목을 '매수'로 올렸습니다.
목표주가 평균은 OOO원. 지금 주가가 OOO원인데... 왜일까요?"
```

### 의문형 후크
```
"[종목명] 지금 사도 되는 걸까요, 팔아야 할까요?
증권사 리포트 핵심만 5분 안에 정리해드리겠습니다."
```

### 역설형 후크
```
"주가가 OO% 빠졌는데, 오히려 목표주가를 올린 증권사가 있습니다.
왜 그런 건지, 오늘 그 이유를 파헤쳐볼게요."
```

### 공감형 후크
```
"[종목명] 검색해보신 분들, 분명 이 질문 때문에 왔을 거예요.
'지금 사도 되나? 아니면 기다려야 하나?'
저도 같은 고민 했거든요. 같이 살펴봅시다."
```

## 섹션 전환 멘트

### 본론 진입
- "자, 그럼 핵심 들어갑니다."
- "핵심 포인트 세 가지, 시작할게요."
- "숫자로 바로 들어가볼게요."

### 섹션 간 전환
- "두 번째 포인트입니다."
- "자, 다음이에요."
- "이제 가장 중요한 부분입니다."

### 리스크 전환
- "좋은 얘기만 하면 섭섭하겠죠. 리스크도 솔직하게 봐야 해요."
- "이 부분은 반드시 알고 넘어가야 합니다."

### 마무리 전환
- "자, 정리해볼게요."
- "오늘 핵심만 다시 짚어볼게요."

## 클로징 문구 모음

### 구독 유도형
```
"오늘 분석이 도움이 됐다면, 구독과 좋아요 꼭 눌러주세요.
매주 [요일]에 새로운 종목 분석 영상이 올라옵니다.
알림 설정도 해두시면 놓치지 않을 수 있어요!"
```

### 블로그 연계형
```
"더 자세한 내용, 표와 수치 정리는 블로그 글에 올려뒀어요.
아래 링크에서 확인해보시고, 투자 결정은 항상 여러분 스스로 하세요!"
```

### 다음 영상 예고형
```
"다음 영상에서는 [다음 종목/주제]를 다룰 예정이에요.
관심 있으신 분들은 구독해두시면 바로 받아보실 수 있습니다!"
```

## 스크립트 전체 예시 (짧은 버전 - 5분 분량)

```
============================================
제목: [종목명] 지금 사야 할까? 핵심 3가지
예상 길이: 5분
============================================

[HOOK - 0~15초]
"[종목명] 목표주가가 OOO원까지 나왔습니다.
지금 주가가 OOO원인데, 상승 여력이 OO%나 된다는 거죠.
과연 근거가 있는 건지, 5분 안에 핵심만 정리해드릴게요."

[채널 소개 - 5초]
"안녕하세요, [채널명]입니다. 오늘 바로 시작합니다."

---

[SECTION 1: 종목 소개 - 30초]
[화면: 회사 로고]
"[종목명]은 [핵심 사업 한 줄 설명]하는 회사예요.
시가총액 OOO조로 코스피/코스닥에서 OO위 규모입니다."

[SECTION 2: 핵심 1 - 1분]
[화면 전환: 슬라이드 - 핵심 포인트 1 제목]
"첫 번째, [핵심 포인트 1 제목]입니다."
[설명 - 구어체로]
[자막: 핵심 수치]

[SECTION 3: 핵심 2 - 1분]
[화면 전환]
"두 번째, [핵심 포인트 2]입니다."
[설명]

[SECTION 4: 핵심 3 - 1분]
[화면 전환]
"마지막 세 번째인데, 이건 리스크예요. 솔직하게 말씀드릴게요."
[설명]

[숫자 정리 - 30초]
[화면: 표 or 인포그래픽]
"숫자 한번 정리하면..."
- 현재가: OOO원
- 목표주가: OOO원

[클로징 - 30초]
"오늘 [종목명] 분석 어떠셨나요?
블로그에도 자세한 글 올려뒀으니 아래 링크 확인해주시고,
구독과 좋아요는 다음 영상을 만드는 큰 힘이 됩니다. 감사합니다!"

[엔딩 카드]
[자막: 본 영상은 투자 권유가 아닌 정보 제공 목적입니다]
============================================
```
