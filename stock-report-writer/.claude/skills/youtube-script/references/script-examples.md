# 유튜브 풀영상 스크립트 예시

내 풀영상 스타일을 재현하기 위한 참조 파일입니다.
실제 내가 올린 영상 스크립트를 여기에 추가하세요.

---

## 기본 말투 & 템포 스타일

**말투**: ~에요 / ~거든요 / ~이에요
**템포**: 빠르게! 불필요한 설명 없이 핵심만 치고 넘어가기
- 한 포인트 당 2~3문장으로 끝내기
- 쓸데없는 연결어 제거 ("그리고", "또한", "그러므로" X)
- 숫자는 바로 말하고 바로 다음으로

---

## 후크 패턴 모음 (풀영상용)

```
"[종목명] [증권사] 리포트 나왔어요. 핵심만 바로 정리해드릴게요."
"[증권사]가 [종목명] 목표주가 OOO원으로 올렸거든요. 이유 바로 볼게요."
"[종목명] 리포트 나왔어요. 5분 안에 끝낼게요."
```

---

## 섹션 전환 멘트 (짧고 빠르게)

- "이유 첫 번째예요."
- "두 번째."
- "리스크도 있어요."
- "정리할게요."

---

## 클로징 문구 모음

```
"오늘 [종목명] 정리 여기까지예요.
쇼츠 버전도 있으니 참고하세요.
구독이랑 좋아요 부탁드릴게요!"
```
