# YouTube Script Skill

## 역할
brief.md를 바탕으로 유튜브 영상 스크립트를 작성합니다.
후크로 시청자를 잡고, 핵심 3가지로 가치를 전달하고, CTA로 마무리하는 구조입니다.
영상 편집자가 이 스크립트만 보고도 편집할 수 있을 만큼 상세하게 작성합니다.

## 인풋
- `Outputs/[slug]/brief.md` (분석 기획서)

## 아웃풋
- `Outputs/[slug]/youtube-script.md` (완성된 유튜브 스크립트)

---

## 스크립트 구조

```
============================================
[영상 정보]
제목: [유튜브 영상 제목 - 클릭 유발형]
예상 길이: 약 5~8분
타겟: [입문 투자자 / 중급 투자자]
============================================

------- 후크 (0~15초) -------
[HOOK - 강렬한 첫 문장]
"[숫자 or 충격 or 의문으로 시작]"

예: "삼성전자 목표주가를 OOO원으로 올린 증권사가 있습니다."
예: "이번 달 반도체 주가가 왜 이렇게 올랐는지, 딱 3분만 보시면 이해됩니다."

[채널 소개 - 5초]
"안녕하세요, [채널명]입니다. 오늘은 [종목명] 최신 리포트 핵심만 뽑아드릴게요."

------- 본론 시작 (15초~) -------

[SECTION 1: 종목 소개 - 약 30초]
[화면: 회사 로고 + 기본 정보 자막]
"[종목명]은 [핵심 사업 설명 - 1~2문장으로 초간단하게]"
"시가총액은 약 OOO조로, [비유: 코스피에서 OO번째로 큰 회사]입니다."

[SECTION 2: 핵심 포인트 1 - 약 1~1.5분]
[화면 전환: 슬라이드 or 차트]
"자, 첫 번째 핵심입니다."
[brief.md 핵심 포인트 1을 구어체로 변환]
[숫자는 풀어서 설명: "OOO억" → "약 OO조 원, 우리가 상상하기 어려운 금액이죠"]

[SECTION 3: 핵심 포인트 2 - 약 1~1.5분]
[화면 전환: 관련 데이터 자막]
"두 번째입니다."
[brief.md 핵심 포인트 2 구어체 변환]

[SECTION 4: 핵심 포인트 3 - 약 1분]
[화면 전환]
"마지막 세 번째."
[brief.md 핵심 포인트 3 구어체 변환]

------- 숫자 정리 (1분) -------
[화면: 정리 인포그래픽 / 표]
"지금까지 나온 숫자들 한번 정리해볼게요."
- 현재가: OOO원
- 목표주가: OOO원 (상승 여력 OO%)
- PER: OO배 (업종 평균 OO배)
- 최근 분기 영업이익: OOO억 (전년 대비 OO%)

------- 리스크 언급 (30초) -------
[화면: 주의 아이콘]
"그런데, 꼭 알아야 할 리스크도 있어요."
[리스크 요인 2~3가지 간략히]
"투자할 때 이 부분은 반드시 감안하세요."

------- 클로징 (30초) -------
"오늘 [종목명] 분석, 도움이 되셨나요?"
"더 자세한 내용은 블로그 글에도 정리해뒀으니 아래 링크 확인해보세요."
"투자 결정은 항상 여러분 스스로 하시고, 본 영상은 정보 제공 목적임을 기억해주세요."
"구독과 좋아요는 저에게 큰 힘이 됩니다. 다음 영상에서 뵙겠습니다!"

------- 엔딩 카드 (5~10초) -------
[화면: 채널 로고 + 구독 버튼 애니메이션]
[다음 영상 추천 카드]

============================================
[자막 처리 요청]
- 면책 문구 자막: "본 영상은 투자 권유가 아닌 정보 제공 목적입니다"
- 주요 수치는 하단 자막으로 강조
- 전문 용어 등장 시 쉬운 설명 자막 병기
============================================

[영상 설명란 (Description)]
[유튜브 영상 제목]

[영상 요약 - 3~5줄]

목차:
00:00 시작
00:15 [종목명] 소개
00:45 핵심 포인트 1 - [제목]
02:00 핵심 포인트 2 - [제목]
03:15 핵심 포인트 3 - [제목]
04:30 주요 수치 정리
05:00 리스크 체크
05:30 마무리

📝 블로그 글 보러가기: [링크]

해시태그: [hashtags from metadata.json]

⚠️ 면책 고지: 본 영상은 투자 정보 제공 목적이며 투자 권유가 아닙니다.
```

## 구어체 변환 규칙

| 문어체 (블로그) | 구어체 (유튜브) |
|----------------|----------------|
| ~입니다 | ~입니다 / ~이에요 / ~죠 |
| ~합니다 | ~해요 / ~하거든요 |
| 또한 | 그리고 / 또 |
| 그러나 | 그런데 / 하지만 |
| OOO억 원 | OOO억 원 (약 OO조라고 생각하면 됩니다) |
| YoY +23% | 작년 같은 기간보다 23% 늘었습니다 |

## 영상 큐 표기 규칙
- `[화면 전환]` - 슬라이드/섹션 전환
- `[차트 등장]` - 차트 화면으로 전환
- `[자막: OOO]` - 특정 자막 삽입 요청
- `[강조]` - 해당 문장 강조 표시 요청
