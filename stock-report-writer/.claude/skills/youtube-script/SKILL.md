# YouTube Script Skill

## 역할
summary.md를 바탕으로 **5~8분 분량의 유튜브 풀영상 스크립트**를 작성합니다.
쇼츠에서 다루지 못한 배경 설명과 맥락을 충분히 담아, 더 깊이 알고 싶은 시청자를 위한 영상입니다.

## 인풋
- `Outputs/[slug]/summary.md`

## 아웃풋
- `Outputs/[slug]/youtube-script.md`

## 아웃풋 형식 (youtube-script.md)

```markdown
# [종목명] 유튜브 스크립트
**예상 시간**: 약 O~O분
**리포트 출처**: [증권사명] ([날짜])

---

## 영상 정보
- **제목**: [클릭률 높은 제목 - 한글 30자 이내]
- **타겟**: 개인 투자자 / 입문 투자자
- **쇼츠 연계**: 쇼츠 영상 링크 설명란에 추가 권장

---

## 스크립트

[전체 스크립트 - 위 구조대로]

---

## 목차 (Description용)

00:00 시작
00:15 [종목명] 소개
00:45 목표주가: OOO원
01:15 이유 1 - [제목]
02:30 이유 2 - [제목]
03:45 이유 3 - [제목] (있을 경우)
05:00 리스크
05:30 정리

---

## 영상 설명란

[영상 제목]

[3~5줄 요약]

목차: (위 목차)

📱 쇼츠 버전도 있어요 → [링크]

본 영상은 리포트를 읽어드리는 것이며 매수, 매도 추천이 아니며 투자에 대한 책임은 본인에게 있습니다.
출처: [증권사명] 리포트 ([날짜])

#[종목명] #주식분석 #증권사리포트 #목표주가 #주식투자

---
*본 영상은 리포트를 읽어드리는 것이며 매수, 매도 추천이 아니며 투자에 대한 책임은 본인에게 있습니다.*
```

## 풀영상 vs 쇼츠 차이

| | 쇼츠 (30~60초) | 풀영상 (5~8분) |
|--|--------------|--------------|
| 목적 | 핵심 한 방 임팩트 | 맥락 + 배경 이해 |
| 종목 소개 | 없음 | 있음 |
| 이유 개수 | 1~2가지 | 2~3가지 |
| 리스크 | 생략 or 1줄 | 충분히 설명 |
| 구어체 수준 | 매우 빠름 | 여유 있게 |
