# 이미지 스타일 가이드

블로그 본문 이미지 생성 시 일관된 브랜드 스타일을 유지하기 위한 가이드입니다.

---

## 색상 팔레트

### 기본 색상 (아래 값을 내 브랜드 컬러로 수정하세요)
```
Primary:   #1E3A5F (네이비 블루)   - 제목, 강조
Secondary: #2E86AB (스카이 블루)   - 서브 요소
Accent:    #F18F01 (오렌지)        - 하이라이트, CTA
```

### 상태별 색상 (금융 차트 표준)
```
상승/긍정: #E74C3C (레드) or #2ECC71 (그린) - 한국은 빨강이 상승
하락/부정: #2ECC71 (그린) or #E74C3C (레드)
중립:      #95A5A6 (그레이)
```

### 배경
```
메인 배경: #FFFFFF (화이트)
카드 배경: #F8F9FA (라이트 그레이)
다크 배경: #1A1A2E (다크 네이비)
```

---

## 폰트 (한국어)

- **제목**: 굵은 고딕체 (예: Noto Sans KR Bold)
- **본문**: 일반 고딕체 (예: Noto Sans KR Regular)
- **수치 강조**: 숫자는 영문 폰트가 더 깔끔함 (예: Inter Bold)

---

## 차트 스타일 가이드

### 바 차트 (실적 비교)
```
- 배경: 흰색
- 바 색상: Primary 색상 계열
- 그리드 라인: 연한 회색 (#E0E0E0)
- 축 레이블: 회색 (#666666)
- 수치 레이블: 바 위에 Bold
- 경계선: 없음 (깔끔하게)
```

### 인포그래픽 카드
```
- 각 지표마다 독립 카드 형태
- 카드 배경: 흰색 + 미세한 그림자
- 수치: 크고 Bold (32px 이상)
- 레이블: 작은 Gray 텍스트 (14px)
- 상승/하락 아이콘: ▲▼ 또는 화살표
```

---

## 텍스트 규칙

- 한국어와 영어 혼용 시 폰트 일관성 유지
- 작은 글씨 최소 크기: 12px (가독성)
- 출처 표기: 우측 하단, 회색 소자 (예: "출처: OO증권 리포트")
- 로고/워터마크: 좌측 하단 또는 우측 하단, 반투명

---

## 생성 프롬프트 공통 요소

모든 이미지 생성 프롬프트에 공통으로 포함:
```
Style: Clean, modern financial infographic
Background: White (#FFFFFF)
Font: Noto Sans KR (Korean), Inter (numbers)
Color scheme: Navy blue (#1E3A5F) as primary
No watermarks from AI
Korean text must be readable
Professional financial design
Resolution: 800x500 pixels
```
