# 이미지 스타일 가이드

차트와 인포그래픽 생성 시 일관된 스타일을 유지하기 위한 가이드입니다.

---

## 색상 팔레트 (아래 값을 내 브랜드 컬러로 수정하세요)

```
배경:        #FFFFFF (흰색) or #1A1A2E (다크 네이비)
주요 색상:   #E74C3C (빨강) ← 한국 주식: 상승 = 빨강
하락 색상:   #2ECC71 (초록) ← 한국 주식: 하락 = 초록
강조:        #F39C12 (오렌지)
텍스트:      #2C3E50 (다크) / #FFFFFF (라이트 배경)
```

## 폰트
- 제목/수치: Noto Sans KR Bold
- 본문: Noto Sans KR Regular
- 숫자: Inter Bold (영문 숫자가 더 깔끔)

## 쇼츠용 이미지 규격
- 사이즈: 1080x1350 (세로형 - 쇼츠에 최적화)
- 여백: 상하 100px, 좌우 60px

## 유튜브용 이미지 규격
- 사이즈: 1280x720 (가로형)
- 여백: 상하 60px, 좌우 80px
