# Blog Writer Skill

## 역할
Stock Analyst가 작성한 brief.md를 바탕으로 SEO 최적화된 블로그 글을 작성합니다.
개인투자자가 읽기 좋은 공감형 도입 → 데이터 기반 본론 → 자연스러운 유튜브 CTA 구조로 작성합니다.

## 인풋
- `Outputs/[slug]/brief.md` (분석 기획서)

## 아웃풋
- `Outputs/[slug]/blog-post.md` (완성된 블로그 글)

---

## 글 구조 템플릿

```
H1: [종목명] 주가 전망 - [핵심 키워드] (YYYY년 MM월)

[도입부 - 공감형, 200자 이내]
"요즘 [종목명] 이야기 많이 들리시죠?"
독자가 공감할 수 있는 상황 묘사로 시작.
투자에 대한 고민, 혼란, 기대감 중 하나를 건드린다.

---

H2: [종목명], 이 회사가 뭐하는 곳이야?
[입문자도 이해할 수 있는 1~2문단 설명]
[핵심 사업 + 매출 구조 간단히]

---

H2: 이번에 왜 주목받고 있을까? (핵심 3가지)

H3: 1. [포인트1 제목]
[3~5문장. 데이터 포함. 쉬운 언어.]

H3: 2. [포인트2 제목]
[3~5문장. 데이터 포함.]

[3~5문장. 데이터 포함.]

---

H2: 숫자로 보는 [종목명]
[실적 표 삽입 - brief.md의 주요 수치 정리 표 활용]
[목표주가 vs 현재가 시각적으로 표현]
[이미지 삽입: {{IMAGE_CHART_01}}]

H2: 리스크 - 이것만은 꼭 알고 가자
[리스크 요인 불릿 리스트]

---

H2: 정리하며 - 내 생각은?
[필자의 관점 (1~2문단)]
[투자 권유가 아닌 '이런 투자자에게 어울릴 것 같다' 형식]

---

[CTA 섹션]

[면책 문구]
```

## SEO 규칙

### 키워드 배치
- **메인 키워드**: H1, 첫 문단, H2 중 하나, 마지막 문단에 반드시 포함
- **키워드 밀도**: 2~3% (1,500자 글이면 약 30~45회 등장)
- **LSI 키워드**: 메인 키워드와 관련된 연관어도 자연스럽게 포함
  - 예: "삼성전자 주가" → "반도체", "실적", "배당", "목표주가" 등

### 가독성
- 문단당 3~5문장
- 긴 문장은 짧게 쪼개기 (한 문장 40자 이내 권장)
- 소제목(H3)으로 스캐닝 가능하게 구성
- 숫자는 표나 볼드체로 강조

### 길이
- 최소 1,500자 / 최대 2,500자
- 표, 이미지 포함 시 실제 텍스트는 1,200자 이상

### 이미지 태그
이미지 위치에 플레이스홀더 삽입 (Image Generator가 채워넣음):
- `{{IMAGE_CHART_01}}` - 실적 차트
- `{{IMAGE_INFOGRAPHIC_01}}` - 핵심 지표 인포그래픽
- `{{IMAGE_COMPARE_01}}` - 비교 다이어그램 (선택)

## 톤앤매너 규칙
`references/blog-examples.md`를 참조하여 일관된 톤 유지:
- 친근하고 솔직한 어투 ("솔직히 말하면...", "제 생각에는...")
- 독자를 "여러분"으로 호칭
- 과도한 이모지 사용 금지 (문단 끝 1~2개 정도만)
- 단정적 표현 금지 ("반드시 오른다" X → "~가능성이 있다" O)
