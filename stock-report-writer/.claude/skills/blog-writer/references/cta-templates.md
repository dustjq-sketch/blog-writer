# CTA 템플릿 모음

블로그 글 마지막에 유튜브 구독 유도 및 내부 링크를 자연스럽게 넣기 위한 CTA 문구 모음입니다.

---

## 유튜브 구독 유도 CTA

### 유형 1 - 영상으로 더 보기
```
> 글로 읽기 귀찮으셨나요? 😅 유튜브에서 영상으로도 같은 내용을 정리해뒀어요!
> 차트와 함께 보면 훨씬 이해가 잘 됩니다.
> 👉 [채널명] 유튜브 구독하기
```

### 유형 2 - 다음 리포트 알림
```
> 다음 주식 분석도 놓치지 않으려면?
> 유튜브 구독 + 알림 설정을 해두시면 새 영상이 올라올 때 바로 받아보실 수 있어요.
> 👉 [채널명] 구독하기
```

### 유형 3 - 간결형
```
📺 영상으로 보고 싶다면 → [채널명] 유튜브
📝 더 많은 종목 분석 → [블로그명]
```

## 내부 링크 CTA

### 관련 종목 연결
```
**관련 글도 확인해보세요:**
- [반도체 섹터 다른 종목명] 분석 보러가기 →
- [동종 업종 경쟁사] 비교 분석 →
```

### 섹터 분석으로 연결
```
**이 종목이 속한 섹터가 궁금하다면:**
→ [섹터명] 전체 분석 글 보러가기
```

## 면책 문구 (필수 포함)

### 표준 면책 문구
```
---
*본 콘텐츠는 투자 정보 제공을 목적으로 하며, 투자 권유가 아닙니다.
모든 투자 결정과 그에 따른 책임은 투자자 본인에게 있습니다.
과거 실적이 미래 수익을 보장하지 않습니다.*
```

### 짧은 면책 문구
```
⚠️ *이 글은 정보 제공 목적이며 투자 권유가 아닙니다. 투자 결정은 본인 책임입니다.*
```

## 사용 원칙
- 유튜브 CTA는 글 마지막, 면책 문구 바로 위에 배치
- 내부 링크는 본문 중간 자연스러운 위치 + 글 마지막 두 곳
- 면책 문구는 반드시 글 최하단에 포함
- CTA가 너무 많으면 스팸스러워 보이므로 2~3개로 제한
