# 쇼츠 스크립트 예시

내 유튜브 쇼츠 스타일을 재현하기 위한 참조 파일입니다.
실제 내가 올린 쇼츠 스크립트를 여기에 추가하세요.

---

## 후크 패턴 모음

### 숫자 직격형 (가장 효과적)
```
"[종목명] 목표주가 OOO원 나왔습니다."
"[증권사]가 [종목명]을 OOO원으로 올렸어요."
```

### 비교형
```
"지금 OOO원인데, 목표주가는 OOO원입니다."
"[종목명], OO% 오를 수 있다는 리포트 나왔어요."
```

### 의문형
```
"[종목명] 지금 사도 될까요? 리포트 나왔습니다."
```

---

## 30초 쇼츠 예시 (75단어)

```
[후크]
"삼성전자 목표주가 10만 원 나왔습니다.
한국투자증권이 오늘 리포트 냈어요."

[본론]
"이유는 두 가지예요.
첫 번째, HBM이라는 AI용 반도체 수요가 폭발하면서
매출이 작년보다 180% 늘었어요.

두 번째, 1분기 영업이익이 3.8조.
증권사 예상보다 20% 더 나왔습니다."

[결론]
"현재가 7만 2천 원 대비 상승 여력 약 39%."

[CTA]
"구독하고 다음 리포트도 받아가세요!"

[면책]
(자막) 투자 권유가 아닙니다
```

## 45초 쇼츠 예시 (110단어)

```
[후크]
"삼성전자, 목표주가 10만 원.
한국투자증권 오늘 리포트 냈습니다."

[종목 한줄]
"반도체 대장주, 시가총액 약 430조 원 규모예요."

[본론]
"이유 두 가지입니다.

첫 번째, HBM이라는 AI 전용 반도체.
일반 메모리보다 5배 비싼데, 올해 매출 비중이 30%를 넘을 전망이에요.
작년보다 180% 늘어난 거거든요.

두 번째, 1분기 실적이 예상보다 20% 높게 나왔어요.
영업이익 3.8조, 시장 예측 3.2조를 훨씬 넘겼습니다."

[결론]
"목표주가 10만 원. 현재 7만 2천 원 대비 39% 상승 여력.
단, 중국 경쟁사 추격은 리스크예요."

[CTA]
"구독하고 다음 리포트도 받아가세요!"

[면책]
(자막) 투자 권유가 아닙니다
```
