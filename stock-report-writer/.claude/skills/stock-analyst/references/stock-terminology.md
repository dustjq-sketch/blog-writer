# 주식 용어 사전 - 독자 친화적 변환

블로그 글과 유튜브 스크립트에서 전문 용어를 쉬운 표현으로 변환할 때 참조하세요.

---

## 밸류에이션 지표

| 전문 용어 | 쉬운 표현 | 추가 설명 |
|-----------|-----------|-----------|
| PER (주가수익비율) | "주가가 순이익의 OO배" | "PER 15배라는 건, 이 회사가 지금 버는 이익의 15배만큼 시장이 값어치를 매긴다는 뜻이에요" |
| PBR (주가순자산비율) | "순자산 대비 주가 수준" | "PBR 1배 미만이면 회사를 청산할 때 받는 돈보다 주가가 싸다는 의미" |
| EV/EBITDA | "기업 전체 가치 대비 영업 현금흐름" | 생략하거나 "(기업가치 평가 지표)" 정도로 처리 |
| ROE (자기자본이익률) | "내 돈으로 얼마나 잘 버는지" | "ROE 20%라면 100만 원 투자해서 20만 원 버는 효율" |
| ROA | "가진 자산 대비 수익성" | |

## 사용 원칙
1. 전문 용어 처음 등장 시 괄호 안에 쉬운 설명 병기: `PER(주가가 순이익의 몇 배인지를 나타내는 지표)`
2. 이후에는 쉬운 표현으로 대체하거나 약어만 사용
3. 유튜브 스크립트에서는 반드시 풀어서 말로 설명
4. 블로그에서는 처음 한 번만 설명하고 이후엔 약어 사용 가능
