# Stock Analyst Skill

## 역할
주식 리포트를 수집하고 핵심을 추출하여 콘텐츠 제작용 분석 기획서(brief.md)를 작성합니다.
복잡한 증권사 리포트를 일반 투자자도 이해할 수 있는 언어로 변환하는 것이 핵심입니다.

## 인풋
- 종목명, 티커
- 리포트 출처 (URL or "웹 검색")
- 분석 기준일

## 아웃풋
`Outputs/[slug]/brief.md` 파일 생성

---

## 실행 방법

### 1. 리포트 수집
우선순위 순서로 수집:
1. 제공된 증권사 리포트 URL 직접 접근
2. 웹 검색: "[종목명] 증권사 리포트 [날짜]"
3. 웹 검색: "[종목명] 주가 분석 [날짜]"
4. 최신 뉴스 3~5개로 대체 (출처 명시)

### 2. 핵심 추출 기준
다음 정보를 반드시 추출:
- **목표주가**: 증권사 제시 목표주가 vs 현재 주가
- **투자의견**: 매수/중립/매도 + 이유
- **실적 요약**: 최근 분기 매출/영업이익 (YoY, QoQ 비교)
- **핵심 드라이버**: 주가 상승/하락 요인 3가지
- **리스크**: 투자 시 주의해야 할 요인

### 3. 용어 변환 규칙
독자 친화적으로 변환 (`references/stock-terminology.md` 참조):

| 전문 용어 | 변환 표현 |
|-----------|-----------|
| PER 15배 | 주가가 순이익의 15배 수준 |
| YoY +23% | 작년 같은 기간 대비 23% 증가 |
| 어닝 서프라이즈 | 시장 예상치를 웃도는 실적 |
| 밸류에이션 | 주가 수준 평가 |
| 컨센서스 | 증권사 평균 예측치 |

## 아웃풋 형식 (brief.md)

```markdown
# [종목명] ([티커]) 분석 기획서
**작성일**: YYYY-MM-DD
**분석 기준**: [리포트명 or 뉴스 출처]

---

## 종목 기본 정보
- **종목명**: [종목명]
- **티커**: [티커]
- **섹터**: [섹터]
- **시가총액**: 약 OOO조 원

## 현재 주가 상황
- 현재가: OOO원
- 목표주가: OOO원 ([증권사명])
- 괴리율: +OO% (목표 대비 상승 여력)
- 52주 최고/최저: OOO / OOO원

---

## 핵심 포인트 3가지

### 1. [가장 중요한 투자 포인트 제목]
[2~3줄 설명. 독자 친화적 언어 사용]
관련 수치: [구체적 숫자]

### 2. [실적 or 이벤트 포인트 제목]
[2~3줄 설명]
관련 수치: [구체적 숫자]

### 3. [리스크 or 주의 포인트 제목]
[2~3줄 설명]
관련 수치: [구체적 숫자]

## 주요 수치 정리

| 항목 | 수치 | 비고 |
|------|------|------|
| 매출 (최근 분기) | OOO억 원 | YoY +OO% |
| 영업이익 | OOO억 원 | YoY +OO% |
| 영업이익률 | OO% | 전분기 대비 OO%p |
| PER | OO배 | 업종 평균 OO배 |
| PBR | O.O배 | |

## 콘텐츠 방향 제안

**예상 검색 의도**: [투자자들이 이 종목을 검색할 때 알고 싶은 것]

**추천 앵글**: [예: "실적 서프라이즈 앵글" / "저평가 발굴 앵글" / "리스크 경고 앵글"]
```

## 주의사항
- 추측성 표현 금지: "오를 것 같다" → "증권사 OO는 OOO원 목표주가 제시"
- 출처 불명확한 수치는 "(추정)" 표시
- 데이터가 부족할 경우 brief.md에 "데이터 부족: [항목]" 명시 후 진행
- `references/stock-terminology.md`를 참조하여 전문 용어는 반드시 쉽게 변환
