# 썸네일 예시 폴더

이 폴더에 기존 블로그/유튜브 썸네일 이미지를 추가하세요.

## 추가 방법
- 기존에 만든 썸네일 이미지를 이 폴더에 복사
- 파일명 형식: `[종목명or주제]-[blog|youtube]-thumbnail.png`
- 예시: `samsung-blog-thumbnail.png`, `nvidia-youtube-thumbnail.png`

## 권장 수량
- 최소 5개 이상 (많을수록 스타일 재현 정확도 높아짐)
- 블로그용 + 유튜브용 골고루

## 왜 중요한가?
Thumbnail Creator Skill이 이 이미지들을 분석하여:
1. 사용하는 색상 팔레트 파악
2. 텍스트 레이아웃 패턴 파악
3. 브랜드 요소 위치 파악
→ 새 썸네일을 같은 스타일로 생성

*썸네일 예시가 많을수록 브랜드 일관성이 높아집니다!*
